# Exercises & Projects - Web Programming (UNSJ)

Welcome to my repository! This space contains practical software systems, web applications, and object-oriented programming (OOP) projects built during my Technical Degree in Web Programming at the **Universidad Nacional de San Juan (UNSJ)**.

## 🛠️ Technologies & Tools
* **Languages:** Python 3.x, HTML, CSS, JavaScript
* **Core Concepts:** Object-Oriented Programming (OOP), Web Development, Data Structures, and System Automation
* **Version Control:** Git & GitHub

## 📂 Featured Projects

* 🍦 **Julito Helados:** A functional web application designed for an ice cream shop, integrating frontend structures with backend logic to simulate commercial operations and user flows.
* 💻 **PO2 2026 (Sistema ETI):** My core academic project for **Práctica Operativa / Programación Orientada a Objetos 2**. Features full functional capabilities for system tracking, advanced data structures, and architecture constraints.
